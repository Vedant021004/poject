# 🧠 NumPy · Pandas · SQL Quiz

A CS50-style terminal quiz. Answer questions, see your grade, results auto-push to GitHub.

---

## 🎬 Video Demo

🔗 **Video Link:** <https://youtu.be/KPSa2wiV09k>

---

## 🚀 How to Run

```bash
git clone https://github.com/Vedant021004/poject.git
cd poject
pip install -r requirements.txt
python project.py
```

---

## 🎮 Controls

| Input | Action |
|-------|--------|
| `A` `B` `C` `D` | Select answer |
| `next` | Skip question |
| `quit` | Exit quiz |

---

## 📚 Topics

- **NumPy** — arrays, broadcasting, indexing, linalg
- **Pandas** — groupby, merge, pivot, loc/iloc, resample
- **SQL** — joins, CTEs, window functions, indexes, ACID

Each topic has **15 questions** across Easy / Medium / Hard.
